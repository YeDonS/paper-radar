# Personalization

Use Zotero-derived signals to boost ranking, but do not let them fully dominate. The point is to bias the radar toward Jun's existing reading taste, not trap it in a tiny bubble.

## Rules

- Treat Zotero high-frequency terms as soft boosts.
- Keep explicit hard-coded system/storage/architecture/AI-infra interests as the main backbone.
- Avoid overfitting to generic words.
- If a Zotero term is suspicious or too generic, drop it.

## Daily digest tiers

Split the digest into:
- 必看: score strongest and clearly aligned
- 可扫: decent fit but weaker novelty / weaker systems value
- 先别看: noisy or only partial alignment
